# Návod na spustenie

## Spustenie databázy

V priečinku projektu spustit príkaz, ktorý pri prvom spustení vytvorý potrebnú schému databázy

```shell
docker-compose up -d
```

## Spustenie aplikácie

Spustenie z konzoly

```shell
java -jar /path_to_jar/swapper.jar
```

Spustenie priamo cez .jar súbor -> dvojklikom na swapper.jar
